# VLSI Full Adder Project

## 📌 Description
This project implements a 1-bit Full Adder using Verilog HDL.  
It is simulated using Icarus Verilog and waveform is viewed using GTKWave.

## 🛠 Tools Used
- Icarus Verilog
- GTKWave

## 📂 Project Files
- adder.v → Verilog design of full adder
- tb_adder.v → Testbench for simulation
- wave.vcd → Waveform output file
